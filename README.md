# 🤖 Ollama AI - Voice Controlled Jarvis

A voice-controlled AI assistant built using Python, Vosk speech recognition, and Ollama.

## 🚀 Features
- 🎙 Voice command recognition (Vosk)
- 🧠 AI responses using Ollama
- 🔊 Text-to-speech output
- 🖥 Runs locally
- 🔐 Offline capable

## 🛠 Tech Stack
- Python
- Vosk
- Ollama
- pyttsx3

## 📦 Installation

```bash
git clone https://github.com/VirajDalvi79/Ollama-ai.git
cd Ollama-ai
pip install -r requirements.txt
python jarvis_voice_controlled.py
```
